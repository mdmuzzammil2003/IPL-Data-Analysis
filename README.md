# 🏏 IPL Data Analysis Project

## Overview
Analysing IPL match data from 2008 to 2022 using Python and Data Analysis libraries.
This project explores team performance, city-wise hosting stats, season trends and player achievements.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
- 950 IPL matches (2008 - 2022)
- 20 columns including teams, cities, winners, toss and players

## Questions Answered
1. Which team won the most matches?
2. Which city hosted the most IPL matches?
3. How many matches were played per season?
4. Who won the most Player of the Match awards?
5. Which team won the most tosses?

## Key Findings
- 🏆 Mumbai Indians won the most matches (~129 wins)
- 🏙️ Mumbai city hosted the most matches (160+)
- 🌟 AB de Villiers won most Player of the Match awards (25)
- 📅 2013 season had the highest number of matches ever
- 🎲 Mumbai Indians also won the most tosses

## How to Run
1. Clone this repository
2. Open ipl_analysis.ipynb in Jupyter or Google Colab
3. Run all cells in order
